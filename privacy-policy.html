수집하는 정보와 앱 권한

WeatherWake는 회원가입·로그인이 없으며 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다. 앱 동작에 필요한 최소한의 권한만 사용합니다.

📍
정밀 위치 (GPS)
현재 위치 날씨를 조회할 때 사용. 이용자가 수동 위치를 선택한 경우 요청하지 않음.
📡
대략적 위치
정밀 GPS를 사용할 수 없을 때 날씨 조회에 활용하는 보조 권한.
🔔
알림 (Notifications)
알람 울림 및 날씨로 인해 알람 시각이 변경되었을 때 알림 표시.
⏰
정확한 알람 예약
설정한 시각에 정확하게 알람이 울리도록 AlarmManager를 사용.
🔄
부팅 완료 수신
기기 재시작 후 설정된 알람을 자동으로 복원하기 위해 사용.
💡
화면 깨우기 (Wake Lock)
알람이 울릴 때 잠금 화면 위에서도 화면이 켜지도록 하는 권한.

3

위치 정보 상세 안내

위치 정보는 현재 날씨를 조회하는 핵심 기능에만 사용되며, 두 가지 방식으로 제공됩니다.

방식 내용 저장 위치
GPS 자동 위치 Google Play Services의 FusedLocationProvider를 통해 현재 기기 위치 조회. 앱 사용 시 및 백그라운드 날씨 체크 시 30분마다 사용. 기기 내 임시 사용 (저장 안 함)
수동 위치 저장 이용자가 직접 입력한 위치(위도·경도·이름). GPS 사용 해제 시 이 값으로 날씨 조회. 기기 내부 SharedPreferences
✅ 위치 정보는 날씨 API 호출 외의 목적으로는 절대 사용되지 않으며, 이용자는 앱 설정에서 언제든 GPS 사용을 해제하거나 저장된 위치를 삭제할 수 있습니다.
4

외부 서비스 연동

WeatherWake는 다음의 외부 서비스를 사용합니다.

서비스 제공사 전송 정보 목적
Current Weather API OpenWeatherMap
(openweathermap.org)		 위도·경도 (좌표만) 실시간 날씨 데이터(비·눈·맑음) 조회
FusedLocation Provider Google Play Services 없음 (기기 내 처리) 기기 GPS 위치 정보 획득
⚠️ OpenWeatherMap의 개인정보 처리 방침은 openweathermap.org/privacy-policy에서 확인할 수 있습니다. 날씨 API 호출 시 위도·경도만 전송되며, 식별 가능한 개인정보는 포함되지 않습니다.
📵 WeatherWake는 광고 SDK, 분석 SDK, SNS 연동 SDK를 사용하지 않습니다. 제3자 광고 네트워크에 어떠한 정보도 제공하지 않습니다.
5

데이터 저장 및 보존 기간

앱의 모든 데이터는 이용자의 기기 내부에만 저장되며, Devkorea1m의 별도 서버에는 저장되지 않습니다.

저장 데이터 저장 위치 내용 삭제 시점
알람 설정 기기 내부 Room DB 알람 시각, 요일, 이름, 날씨 연동 설정, 알람음 경로 이용자가 알람 삭제 또는 앱 삭제 시
저장 위치 기기 SharedPreferences 위도·경도 좌표, 위치 이름, GPS 사용 여부 이용자가 위치 초기화 또는 앱 삭제 시
날씨 결과 메모리 (임시) 현재 날씨 상태 (비/눈/맑음), 날씨 설명 앱 프로세스 종료 시 자동 삭제
💡 앱을 기기에서 삭제(언인스톨)하면 위의 모든 로컬 데이터가 완전히 삭제됩니다. 클라우드 백업(android:allowBackup)은 시스템 백업에만 한정되며, Devkorea1m은 이 데이터에 접근할 수 없습니다.
6

제3자 제공 및 공유

Devkorea1m은 이용자의 개인정보를 제3자에게 판매·임대·제공하지 않습니다.

  • ① 날씨 조회를 위해 위도·경도 좌표가 OpenWeatherMap API로 전송됩니다. 이는 서비스의 핵심 기능 제공을 위해 필수적이며, 식별 가능한 개인정보가 포함되지 않습니다.
  • ② 법령의 규정에 의거하거나 수사기관의 적법한 요청이 있는 경우 법이 허용하는 범위 내에서 최소한으로 제공할 수 있습니다.
  • ③ 그 외의 경우에는 이용자의 명시적 동의 없이 개인정보를 제3자에게 제공하지 않습니다.
7

이용자의 권리

이용자는 앱 내에서 직접 아래 권리를 행사할 수 있습니다.

  • ✓ 위치 정보 삭제 — 앱 위치 설정 화면에서 저장된 위치를 삭제할 수 있습니다.
  • ✓ GPS 권한 철회 — 기기 설정 → 앱 권한에서 위치 권한을 언제든지 거부할 수 있습니다. (GPS 모드 사용 불가)
  • ✓ 알람 데이터 삭제 — 앱 내에서 개별 알람을 삭제하거나, 앱 삭제 시 모든 데이터가 완전히 삭제됩니다.
  • ✓ 백그라운드 동작 제어 — 기기 설정에서 앱의 백그라운드 활동을 제한하여 30분 주기 날씨 체크를 비활성화할 수 있습니다. (알람 정확도에 영향)

그 외 개인정보 열람·정정 요청은 아래 담당자 이메일로 문의하시면 영업일 기준 3일 이내 처리합니다.

8

보안 조치

🔒 HTTPS 통신 강제 적용 📦 로컬 DB 앱 샌드박스 격리 🚫 광고/트래킹 SDK 미사용 🔑 API 키 앱 내 관리

WeatherWake의 모든 네트워크 통신은 HTTPS를 통해 암호화되어 전송됩니다. 기기 내 저장 데이터는 Android 앱 샌드박스로 보호되어 다른 앱이 접근할 수 없습니다.

날씨 API 키는 앱 내부에 포함되어 있으며, 이용자의 데이터 전송에는 사용되지 않습니다.
