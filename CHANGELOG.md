## v1.2.0 (2026-04-14)

### 새 기능
- **OpenWeatherMap Attribution 표시** — 메인 날씨 카드 하단에 "Weather data by OpenWeatherMap" 크레딧 추가 (탭 시 OpenWeatherMap 홈페이지 열림)

- **Android 15/16 대응** — Edge-to-edge 처리를 `enableEdgeToEdge()` API로 전환 (지원 중단된 `setDecorFitsSystemWindows` 제거)
- **대형화면 호환성** — AlarmRingActivity의 세로 방향 고정 제거. 태블릿·폴더블에서도 정상 작동

## v1.1.0 (2026-04-14)

### 새 기능
- **날씨 설정 전용 화면 분리** — 비·눈 설정을 각각 독립된 화면(2단계, 3단계)으로 분리. 한 화면에 스크롤 없이 표시
- **4단계 민감도 선택** — 아주 민감하게 / 민감하게 / 보통(권장) / 둔감하게
- **알람 카드 날씨 요약 표시** — 날씨 연동 ON 시 민감도·앞당김 시간을 알람 카드에서 바로 확인 가능

### 개선
- 비 감지 기준을 한국 체감에 맞게 조정 (보통 기준 2.5 → 1.0 mm/h)
- 날씨 설정 화면 글자 크기 개선으로 가독성 향상
- 날씨 체크 방식 안내 문구 추가 (알람 시각 90분 전 자동 확인)
- 알람 카드의 앞당김 표시 "1h" → "1시간" 통일

### 버그 수정
- 화면 회전 시 날씨 설정(민감도·앞당김 시간)이 초기화되던 문제 수정
- 이슬비(안개비) 감지 오작동 수정 — OpenWeatherMap이 강수량 데이터를 생략하는 경우에도 아주민감·민감 설정 시 정상 트리거
